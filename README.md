# ⛏️ Stone Farming Bot (FiveM)

บอทขุดหินอัตโนมัติสำหรับ FiveM — เมื่อ Stone เต็ม 100/100 จะฝากเข้าท้ายรถให้อัตโนมัติ แล้วกดออโต้ฟาร์มต่อ วนลูปไปเรื่อย ๆ

> ⚠️ ใช้กับเซิร์ฟที่อนุญาตเท่านั้น การใช้บอทอาจผิดกฎเซิร์ฟและเสี่ยงถูกแบน — ผู้ใช้รับความเสี่ยงเอง

## สิ่งที่ต้องมี
- Windows + Python 3.10 ขึ้นไป ([ดาวน์โหลด](https://www.python.org/downloads/) ตอนติดตั้งให้ติ๊ก **Add Python to PATH**)
- จอ 16:9 (720p / 1080p / 1440p / 4K — รองรับอัตโนมัติ)

## ติดตั้ง
```bash
pip install -r requirements.txt
```

## วิธีใช้
1. **รันด้วยสิทธิ์ Administrator** (จำเป็น ไม่งั้นส่งปุ่มเข้าเกมไม่ได้)
2. ครั้งแรก — จับ template ให้ครบ:
   ```bash
   python calibrate.py
   ```
   - เปิดกระเป๋าในเกม ชี้เมาส์กลางไอคอน Stone → กด `1`
   - ชี้จุด/ปุ่มต่าง ๆ ตามที่โปรแกรมบอก → กด `2`-`5`
   - เปิดแผนที่/เมนู pause ค้างไว้ → กด `6`
   - เปิดหน้า GARAGE (กด L) ค้างไว้ → กด `7`
   - กด `0` เพื่อบันทึกและออก
3. รันบอท:
   ```bash
   python stone_main.py
   ```
4. สลับไปหน้าเกม (ยืนใกล้รถให้กด L เปิดท้ายรถได้) → กด **F10** เริ่ม/หยุด → **Esc** ในคอนโซลเพื่อปิด

## Flow
```
ออโต้ฟาร์ม (G) → อ่านเลข Stone จาก HUD
  → เต็ม 100/100 → กด L → เปิดหลังรถ → ลาก Stone → Max → O → ESC → G → วนต่อ
```

## ปรับแต่ง
แก้ค่าทั้งหมดได้ใน [`config.py`](config.py) — ดีเลย์แต่ละขั้น, เกณฑ์ตรวจจับ, จำนวนครั้ง retry

## กันพัง (ในตัว)
- เจอแผนที่/เมนูเปิดค้าง → กด ESC ปิดแล้วฟาร์มต่อ
- กด L แล้วหน้า GARAGE ไม่เปิด → กด L ซ้ำ
- หาช่อง Stone ไม่เจอ → เลื่อน inventory ขึ้นแล้วหาใหม่
- ทำพลาดหลายครั้งติด → หยุดบอท + บันทึกภาพใน `debug_images/`

## ไฟล์
| ไฟล์ | หน้าที่ |
|------|--------|
| `stone_main.py` | loop หลัก + F10 toggle |
| `stone_detector.py` | อ่านเลข Stone + หาช่อง + ตรวจเมนู |
| `stone_actions.py` | ลำดับฝากของเข้าท้ายรถ |
| `stone_input.py` | ส่งปุ่ม/เมาส์ (DirectInput) |
| `config.py` | ค่าพิกัด/ดีเลย์/เกณฑ์ |
| `calibrate.py` | จับพิกัด + template ของจอตัวเอง |
